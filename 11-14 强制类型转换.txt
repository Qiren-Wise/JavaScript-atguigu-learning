P11. 其他类型转换为字符串
方式一：
    - toString()方法
    - 不影响原变量
    - null和undefined不能使用本方法，会报错
方式二：
    - 调用String()函数
    - null和undefined允许使用本方法，还是等于null和undefined。

P12. 其他类型转换为数值型
方式一：
    - 调用Number()函数
    - 字符串 --> 数值
        # 数字可以直接转
        # 带有非数字字符会转换成NaN
        # 空字符串转换为0   
    - 布尔 --> 数值
        # true = 1
        # false = 0
    - nall --> 数值 = 0
    - undefined --> 数值 = NaN
方式二：
    - 专门对付字符串
    - parseInt() 转换为整数
        # 可以将字符串中的有效整数内容读取出来，所以非数值类型不会读取
        # 从左往右只读取第一个非数值字符串前的整数，后面的就读取不到了
    - parseFloat() 转换为浮点数
        # 和上面同理，只不过这个是保存小数
    - 布尔 --> 数值
        # 使用本方式会返回NaN
        # 因为本方式会先将True和False转换为字符串，'True'不是数字

P13. 其他进制的数字
16进制 --> 在js中以0x开头
8进制 --> 在js中以0开头
    - 如'070'这种数字有些浏览器会当成八进制解析，有些浏览器会当成10进制解析
    - 可以在parseInt()中加入第二个参数，第二个参数为8则八进制，10则十进制。如ParseInt('070',10); 这样就会统一为10进制解析
2进制 --> 在js中以0b开头
    -但并不是所有浏览器都支持

P14. 其他类型转换为布尔值
使用Boolean()函数
    - 数字 --> 布尔
        # 除了0和NaN，都是返回True
    - 字符串 --> 布尔
        # 除了空字符串，都是返回True(有空格也是返回True)
    - null和undefined都是转成False
    - 对象也会转换为True
